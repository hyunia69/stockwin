Endpoints

GET /stockwin_get_arstype.asp

ARS 전화번호에 배정된 결제유형을 조회합니다.

Parameter Type Required Description
mode string 필수 인증 모드 (고정값)
ars_tel_no string 필수 ARS 전화번호 (4자리)
Response Format
ars_tel_no|type_cd_list|Error_cd|message
Example Response
2972|CQS|0000|결제유형획득에 성공하셨습니다.
POST /stockwin_item_update.asp

결제 시나리오를 등록하거나 수정합니다.

Parameter Type Required Description
mode string 필수 인증 모드 (고정값)
shop_id string 필수 arsstockwin | arsstockwin1 | arsstockwin2
ars_tel_no string 필수 ARS 전화번호 (4자리)
scenario_type string 필수 CQS | CIP
arrribute_type string 필수 1MONTH | 1WEEK | 2WEEK | NOFREE | ADVRESERVD | SERVICE | EDUCATION
amount string 필수 결제/안내 금액
cc_pord_desc string 필수 상품명^상품코드
startdtm string 조건부 ADVRESERVD일 때 필수 (YYYYMMDDHHMM)
shop_id 유형
arsstockwin2
정기결제 (빌키 발급)
arsstockwin1
간편결제
arsstockwin
일반결제
Response Format
ars_tel_no|Item_cd|Error_cd|message
Example Response
2972|PROD001|0000|등록이 완료되었습니다.

에러 코드

Code Description
0000 성공
0001 mode 값 오류
0002 ars_tel_no 또는 shop_id 누락
0003 ars_tel_no 누락
0004 scenario_type 오류
0005 arrribute_type 누락
0006 amount 누락
0007 cc_pord_desc 누락
0008 startdtm 누락 (ADVRESERVD)
9001 데이터베이스 오류
9002 전화번호 미등록
