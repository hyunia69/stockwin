# Part 1: 멘트 요약 (텍스트만)

## 전체 흐름 요약

```
인사말 → 고객식별 → 동의서 → 결제정보 안내 → 카드입력 → 유효기간 → 할부선택 → 결제처리 → 종료
```

## 4. 결제금액 및 해지조건 안내

### 4-1. 결제금액 안내

#### 쿠폰 + 보너스캐시 적용
> **[TTS]** "{고객명} 고객님, {가맹점명}에서, 주문하신 {상품명}의 결제 금액은 {고객명}님께서 보유하신 {쿠폰명} 쿠폰과 보너스 캐시 {금액}원이 적용되어 최종 결제 금액은 {결제금액}원입니다."

#### 쿠폰만 적용
> **[TTS]** "{고객명} 고객님, {가맹점명}에서, 주문하신 {상품명}의 결제 금액은 {고객명}님께서 보유하신 {쿠폰명} 쿠폰이 적용되어 최종 결제 금액은 {결제금액}원입니다."

#### 보너스캐시만 적용
> **[TTS]** "{고객명} 고객님, {가맹점명}에서, 주문하신 {상품명}의 결제 금액은 {고객명}님께서 보유하신 보너스 캐시 {금액}원이 적용되어 최종 결제 금액은 {결제금액}원입니다."

#### 할인 없음 (기본)
> **[TTS]** "{고객명} 고객님, {가맹점명}에서, 주문하신 {상품명}의 결제하실 금액은 {결제금액}원입니다."

### 4-2. 해지조건 안내 (상품유형별)

#### 태블릿 제공 상품 (TABLET)
> **[TTS]** "박스 개봉 후 제품 불량을 제외하고는 교환 및 반품이 불가합니다."

#### 교육상품 (EDUCATION)
> **[TTS]** "본 상품은 교육 상품으로 결제 후 해지가 불가능합니다. 또한, 무료로 제공되는 서비스는 중도 해지 및 일시 정지 파일 양도가 불가합니다."

#### 일반상품 (SERVICE) - 기본
> **[TTS]** "서비스 중도해지 시 해지일까지 이용요금과 해지수수료 10퍼센트와 제공받으신 사은품 정가가 함께 차감됩니다."

### 4-3. 쿠폰/보너스캐시 소멸 안내 (SERVICE 상품 + 할인 적용 시)

#### 쿠폰 + 보너스캐시 사용
> **[TTS]** "또한, 자동 결제 시 적용되는 {쿠폰명} 쿠폰과 보너스캐시 {금액}원은 소멸됨을 알려드립니다."

#### 쿠폰만 사용
> **[TTS]** "또한, 자동 결제 시 적용되는 {쿠폰명} 쿠폰이 소멸됨을 알려드립니다."

#### 보너스캐시만 사용
> **[TTS]** "또한, 자동 결제 시 적용되는 {금액}원 보너스캐시는 소멸됨을 알려드립니다."

### 4-4. 투자 유의사항 (공통)
> **[TTS]** "또한, 한국경제티비와, 파트너는, 금융투자업자가 아닌, 유사투자자문업자로, 개별적인 투자 상담과 자금 운영이 불가하며, 원금 손실이 발생할 수 있고, 그 손실은 투자자에게 귀속됩니다."

### 4-5. 최종 동의 확인
> **[TTS]** "동의 및 결제하시려면 1번을, 취소하시려면 2번을 눌러주세요."

# Part 2: 상세 시나리오 흐름 (파일경로 포함)

## 시나리오 타입
| 타입 | 설명 |
|------|------|
| **ARS** | 고객이 전화번호 직접 입력 |
| **SMS** | SMS로 받은 주문번호 입력 |
| **CID** | Caller ID로 자동 조회 |
| **CIP** | Caller ID 우선 → 휴대폰 아니면 전화번호 입력 |
